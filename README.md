
## Cloning the project
```bash
$ git clone https://github.com/tushermahmud/giftcard.git
```
## Installation
# Go to client folder and run 
```bash
$ cd client && npm install

```
# Go back to  
```bash
$ cd ..&&npm install

```

## Running the app

```bash
$ npm start

```
